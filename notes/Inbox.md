# Inbox

Capture rapide d'idées et de tâches. Traitez régulièrement cette boîte.

> [!tip] Quick Capture
> Utilisez **Win+Shift+N** pour capturer rapidement une idée depuis n'importe où.

---

## Traitement Inbox

**Règles de traitement:**
1. Transformer en tâche actionnable → ajouter dans Daily note
2. Idée à développer → créer note dédiée
3. Référence → archiver dans note appropriée
4. Obsolète → supprimer
